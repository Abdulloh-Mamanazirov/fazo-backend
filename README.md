## Installation

```bash
# npm
$ npm install

# pnpm
$ pnpm install

# yarn
$ yarn
```

## Running the application

---

**with** `pnpm`

```bash
# run the application
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

---

**with** `npm`

```bash
# run the application
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

---

**with** `yarn`

```bash
# run the application
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
